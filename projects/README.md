# Projects

Active feature work lives here as **project capsules** — one folder per feature,
each with a required `PLAN.md` front door.

- New capsule: `projects/<slug>/PLAN.md` (copy the template from
  `docs/agent-process/project-capsules.md`).
- Done or abandoned capsules move to `projects/_finished/<YYYY-MM-DD>-<slug>/` or
  `projects/_archive/<YYYY-MM-DD>-<slug>/`.

See **`docs/agent-process/project-capsules.md`** for the full convention.
