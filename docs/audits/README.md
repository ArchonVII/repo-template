# docs/audits — unresolved human corrections

High-priority corrections and known conflicts live here as individual files. Review them at
session start (`npm run wiki:start` surfaces them). Resolve by updating the affected page,
then move the audit file to `docs/audits/archive/`. **Never ignore an open audit.** This tier
is not linted for wiki frontmatter (see [docs/LIBRARIAN.md](../LIBRARIAN.md) "What is NOT a page").
