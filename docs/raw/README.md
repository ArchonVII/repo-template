# docs/raw — immutable intake sources

Drop human-curated source material here (notes, transcripts, exported docs). The Librarian
**reads** these and never rewrites them. Run `npm run wiki:ingest -- docs/raw/<file>` to start
an ingest. This tier is not linted for wiki frontmatter (see [docs/LIBRARIAN.md](../LIBRARIAN.md)
"What is NOT a page").
